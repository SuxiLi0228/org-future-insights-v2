- 基于 Vite + React 18 的单页应用架构，入口为 index.html 加载 src/main.tsx。
- 使用 @vitejs/plugin-react 支持 JSX/TSX，配置 @ 别名指向 src 目录简化模块引用。
- 样式采用 Tailwind CSS 原子化方案，通过 tailwind.config.js 定义品牌色板、字体及动画关键帧。
- 路由采用客户端渲染模式，vercel.json 配置将所有路径重写至 index.html 以支持前端路由。
- 构建流程集成 TypeScript 类型检查与 Vite 打包，输出至 dist 目录并生成 sourcemap。
